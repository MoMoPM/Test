Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Registration Req

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"nwsrlntlowjfxuibzpuppczgrxare\",
    \"email\": \"isaiah.kunze@example.net\",
    \"password\": \"m2s-]4}}JuOU7~,\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "nwsrlntlowjfxuibzpuppczgrxare",
    "email": "isaiah.kunze@example.net",
    "password": "m2s-]4}}JuOU7~,"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Store a newly created resource in storage.

Example request:
curl --request POST \
    "http://localhost/api/movies" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "title=ijmvhz" \
    --form "summary=x" \
    --form "rating=3" \
    --form "link=fy" \
    --form "image=@/private/var/folders/7y/5gb59l0n0_nc3hh9y9gy0jvr0000gp/T/phpcRqzfe" 
const url = new URL(
    "http://localhost/api/movies"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('title', 'ijmvhz');
body.append('summary', 'x');
body.append('rating', '3');
body.append('link', 'fy');
body.append('image', document.querySelector('input[name="image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
